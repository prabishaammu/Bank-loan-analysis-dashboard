Bank Loan Analysis Dashboard - Power BI

Objective
Analyze bank loan data to identify good vs bad loans, 
monthly trends, and loan distribution across states, 
purposes and employment lengths.

Dataset
- Source: Financial Loan Dataset (Kaggle)
- Records: 39,000+ rows
- Columns: 24 columns

Tools Used
- Power BI Desktop
- Power Query
- DAX

Key Metrics
- Total Applications: 39K
- Total Funded Amount: $436M
- Total Amount Received: $473M
- Avg Interest Rate: 12.05%
- Avg DTI: 13.33%

Key Insights
- 86% Good Loan rate vs 14% Bad Loan rate
- Debt consolidation is the most common loan purpose
- Loan applications grew consistently month over month
- California and New York are top states for loan applications
- 60 month term loans have higher funded amounts

DAX Measures Created
- Total Applications
- Total Funded Amount
- Total Amount Received
- Avg Interest Rate
- Avg DTI
- Good Loan %
- Bad Loan %
